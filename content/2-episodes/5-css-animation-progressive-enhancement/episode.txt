Summary: 

When we use animation to bring content into view we can end up with content that’s never visible if animations aren’t supported. That’s not cool! In this episode, I show you a way to avoid this using progressive enhancement and a cute cat. The example is live on [codepen](cdpn.io/IodKn) if you’d like to follow along.
